# Citation Policy

Accepted public artifacts require source metadata.

Downstream users should normally work from
[`tcs-cosheaf-workspace-template`](https://github.com/CheemsaDoge/tcs-cosheaf-workspace-template),
which mounts this public KB readonly and keeps private notes in a separate
writable overlay. This citation policy applies to accepted public artifacts in
this repository; private workspace drafts are not a substitute for public source
metadata.

## Source Expectations

- Prefer stable, public, citable sources.
- Record bibliographic notes under `sources/` when local source metadata is needed.
- Use external evidence paths only for broad standard terminology or when the source is intentionally not vendored.
- Keep source summaries concise and factual.

## Review Expectations

Before an artifact is accepted, reviewers should confirm that:

- the statement matches the cited source;
- dependencies are public artifacts only;
- source metadata is complete enough to cite and relocate the source;
- the artifact does not include private notes or unpublished claims;
- validation and gate results are recorded or explicitly explained.
